# 🇬🇧 Unit Converter 🇬🇧

*Solo project created during JavaScript course at [Scrimba](scrimba.com).*

Quickly allows you to convert metric units to imperial units and vice versa. 

- Length (Meter/Feet)
- Volume (Liters/Gallons)
- Mass (Kilograms/Pounds)
